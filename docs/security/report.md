# 安全报告

> 安全策略详见根目录 [SECURITY.md](../../SECURITY.md)（英文）

## 报告漏洞

请通过以下渠道联系维护者：

1. **GitHub Security Advisory**：https://github.com/HyperSynapseNetwork/Phira-mp-plus/security/advisories
2. **邮箱**：security@phira-mp-plus.dev

预计 48 小时内回复。如未收到，请跟进。

## 处理流程

1. 收到报告 → 48 小时内确认
2. 调查 → 开发修复（时间取决于严重度）
3. 修复发布 → 协调披露

## 范围

- PMP 游戏服务端（本仓库）
- 官方 WASM 插件 SDK
- Phira+ 协议实现

**不包括**：第三方插件、自定义部署、Phira 前端。
